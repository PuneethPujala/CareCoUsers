CareMyMed Flow.io Pipeline Diagram

Interactive Architecture & Data Integrity Map

HIGHLIGHT PATHWAY:
1. Client Inputs
2. Offline & State Cache
3. API Gateway
4. Analytics & Engines
❤️

Vitals Logging

Patient Input
User captures SpO2, heart rate, and blood pressure logs. Uses local timezone boundaries dynamically.
Model: VitalLog Client
💊

Meds Compliance

Patient Input
Ticks off scheduled daily doses. Empty meds plans default to null to protect baseline stats integrity.
Model: MedicineLog Client
🌙

Sleep Tracker

4-Tier Estimation
Hierarchy: 1. Health Connect (Wearable), 2. UsageStatsManager (Screen activity Z-Scores), 3. Manual, 4. Amber CTA.
Model: SleepLog Client
😊

Mood Check-in

Patient Input
Captures daily qualitative state (sad, okay, good, great) mapping to numbers (1-4) for trend vectoring.
Schema: Patient.mood Client
📦

Zustand State Store

usePatientStore
Client-side single source of truth. Dispatches optimistic UI updates and synchronizes cached state variables.
Context: React Local Cache
🔄

Offline Sync Queue

OfflineSyncService
Serializes pending updates to AsyncStorage when offline. flushes FIFO queries once connectivity resumes.
Storage: AsyncStorage Local Queue
⚡

Express Server API

Router Gateway
Authorizes tokens via dual JWTs/Supabase middleware. Filters queries by scope, parses timezones, mounts routes.
Server: Express.js Port 3001
⏰

BullMQ Queue

5s Delay Deduplication
Queues recomputations to prevent database hammering on multiple writes. Synchronizes worker tasks.
Queue: BullMQ Redis Backed
📊

Health State Engine

Score Recomputation
Runs personalBaselineService Z-Scores. Adjusts base score for anomalies & appends tips.
Service: stateService Aggregator
💾

Mongoose DB & Redis

Persistence & Cache
Saves state to Patient document. Upserts daily record snapshots to PatientHealthStateHistory.
DB: MongoDB 60-Day Expire
🏁

35-Day Balance Grid

StreakCalendar UI
Grids are colored strictly based on overall health score (Whole Health). Hides score / empty details on unlogged days.
Component: StreakCalendar Client UI
